# 📊 NIFTY 100 Analytics Dashboard

An end-to-end **Financial Analytics & Equity Research Platform** built using Python, SQL, SQLite, Pandas, and Streamlit.

The project extracts, transforms, validates, and analyzes NIFTY 100 company financial data to generate investment insights through interactive dashboards.

---

# 🚀 Project Overview

The NIFTY 100 Analytics Dashboard provides a complete workflow:

The platform helps analyze:

- Company performance
- Financial ratios
- Growth trends
- Peer comparison
- Sector performance
- Capital allocation
- Research reports

---

# 🎯 Objectives

- Build a reliable financial data pipeline
- Automate financial metric calculations
- Perform equity research analysis
- Create interactive dashboards
- Generate company-level insights
